# ProctorAI

Privacy-preserving exam and browser-side AI proctoring platform.

## Quick Start

1. Install root deps

```bash
npm install
```

2. Install backend deps

```bash
cd server
npm install
```

3. Install frontend deps

```bash
cd ../client
npm install
```

4. Configure environment files

- Copy `server/.env.example` to `server/.env`
- Copy `client/.env.example` to `client/.env`

5. Run dev servers from root

```bash
npm run dev
```

- Client: http://localhost:5173
- Server: http://localhost:5000
